# InvOperaciones-EjerciciosDeOptimizacion
Aplicación web interactiva desarrollada en HTML, CSS y JavaScript que resuelve 6 problemas de optimización lineal y enter mixta (MILP) relacionados con el departamento del Cauca, Colombia.  Incluye ejercicios ordenados por dificultad: • 2 Básicos • 2 Intermedios • 2 Avanzados Desarrollado para la materia Investigación de Operaciones - Unicomfacauca
