# 📊 Optimización Operaciones - Cauca

Aplicación web interactiva para la resolución de problemas de optimización lineal y programación entera mixta (MILP), contextualizados en el departamento del Cauca, Colombia.

---

## 🎯 Objetivo

Desarrollar una interfaz gráfica intuitiva que permita visualizar y resolver problemas de Investigación de Operaciones, comparando los resultados obtenidos en **Python (PuLP)** con su implementación en **JavaScript (jsLPSolver)**.

---

## 📚 Problemas incluidos

| Nivel | Ejercicio | Tipo | Descripción |
|-------|-----------|------|-------------|
| **Básico 1** | Producción Artesanal | LP Maximización | Canastos y mochilas del Cauca |
| **Básico 2** | Velas y Jabones | LP Maximización | Productos naturales ecológicos |
| **Intermedio 1** | Bodegas de Alimentos | MILP Minimización | Localización de bodegas para municipios |
| **Intermedio 2** | Centros de Salud | MILP Minimización | Distribución de medicamentos en hospitales |
| **Avanzado 1** | Tecnología PDET | MILP Minimización | Distribución multiperíodo de equipos educativos |
| **Avanzado 2** | Lot Sizing | LP Minimización | Planificación de producción con inventarios |

---

## 🛠️ Tecnologías

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Solver:** [jsLPSolver](https://github.com/JWally/jsLPSolver) — Librería pura de JavaScript para programación lineal
- **Diseño:** Paleta de colores azules, interfaz responsive con navegación lateral
